# 📊 Sales Dashboard Analysis

## 📌 Project Overview

This project presents an interactive sales dashboard built using Python and Tableau Public. It analyzes sales performance across different categories, products, and time periods to generate meaningful business insights.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Tableau Public

---

## 📈 Dashboard Features

- Total Sales KPI
- Monthly Sales Trend
- Category-wise Sales Distribution
- Top 10 Products by Sales
- Interactive Dashboard

---

## 📂 Repository Contents

- `SalesDataAnalysis.ipynb` – Data cleaning and analysis
- `Sales_data.csv` – Dataset
- `Sales Data Analysis.twbx` – Tableau Dashboard

---

## 🚀 How to Run

1. Download this repository.
2. Open `SalesDataAnalysis.ipynb` in Jupyter Notebook.
3. Open `Sales Data Analysis.twbx` using Tableau Public Desktop.
